# Collapsible Section

A collapsible section can go by many names and is usually refered to as an _accordion_. _Accordion_ implies more than one whereas _Collapsible Section_ can be one or more.

## Usage

```
<CollapsibleSection heading="My Accordion Heading">
  <p>
    Ipsum sequi non earum dolores quos. Odit hic saepe nam eveniet error
    voluptatem saepe. Temporibus excepturi ipsam repellat recusandae a Harum
    deleniti quo esse quidem facilis velit! Voluptates voluptatum nesciunt
  </p>
</CollapsibleSection>
```

If designs require the collapsible section to be open by default, adding the `open` prop will make that happen.

## Props

| Name       | Required | Type                                 | DefaultValue |
| ---------- | -------- | ------------------------------------ | ------------ |
| heading    | yes      | string                               |              |
| preHeading |          | string                               |              |
| children   | yes      | React.ReactNode \| React.ReactNode[] |              |
| open       |          | boolean                              | false        |
