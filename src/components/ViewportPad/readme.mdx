# Viewport Pad

Viewport pad is a centering utility that ensures content is a minimum safe distance from the left and right of the viewport. It is commonly used with other layout utilities.

## Usage

```
<ViewportPad>
  <div className="fpo">Content goes here.</div>
</ViewportPad>
```

## Props

| Name     | Required | Type                                 | DefaultValue |
| -------- | -------- | ------------------------------------ | ------------ |
| children | yes      | React.ReactNode \| React.ReactNode[] |              |
