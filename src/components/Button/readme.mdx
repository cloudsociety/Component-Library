# Button

## Usage

```
<Button
    type="button"
    secondary
    buttonStyle="hollow"
>
  Click me
</Button>
```

## Props

Props are extended from `ButtonHTMLAttributes`. This means all standard button attributes can be passed to the component as props.

In addition to the standard attributes, you can use these:

| Name         | Required | Type                           | DefaultValue |
| ------------ | -------- | ------------------------------ | ------------ |
| secondary    |          | boolean                        | false        |
| large        |          | boolean                        | false        |
| buttonStyles |          | "normal" \| "hollow" \| "link" | "normal"     |
| classes      |          | string                         | ""           |
